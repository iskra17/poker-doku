# Poker Now 딥 리서치 (2026-07-18)

경쟁 플랫폼 Poker Now(pokernow.club → pokernow.com) 심층 분석 — 기능·강점·단점·커뮤니티 반응을
조사해 포커 도쿠의 벤치마킹 후보와 차별화 축을 도출한다.

- **방법**: 딥 리서치 워크플로 — 5개 앵글 검색 팬아웃 → 소스 20개 정독 → 주장 99건 추출 →
  상위 25건 각 3표 적대 검증(22 확정 / 3 반박 폐기 / 0 미검증), 에이전트 102개.
- **기준일**: 가격·혜택·이슈 표수는 전부 2026-07-18 라이브 확인 기준. 시간이 지나면 재확인할 것.

## 1. Poker Now 개요

무가입·무설치 브라우저 홈게임 플랫폼. 방 생성은 블라인드·스택·변형 설정 후 즉시 완료되는
셀프서브이고, 초대는 룸 링크 공유(문자/WhatsApp/Discord) — "No sign-up needed. Anyone with the
room link can join and play in seconds". 브라우저 전용이 "part of our DNA"(공식 FAQ). 폰·태블릿·
데스크톱에서 수 초 내 참가. 좌석 요청은 호스트 승인 필요, 토너먼트·구독 등 일부 기능은 가입 요구.

### 기능 폭 (확정된 사실)

| 영역 | 내용 |
|---|---|
| 턴 타이머 | 기본 20초, '무제한' 옵션까지 커스터마이즈 가능 |
| 블라인드 | 테이블당 최대 **30레벨**, 레벨별 SB/BB/지속시간(분) 개별 설정 |
| 토너먼트 | 싱글테이블 + **MTT 최대 300명**, 사전 체크인(2023-09), 초대코드 CSV 관리·자동 승인(2022-05), Plus 회원 30분 레이트 레지 |
| 변형 룰 | run it twice(2021-01)·rabbit hunting(2021-02)은 무료 계열, Double Board·Bomb Pots·7-2 Bounty·NIT Game은 **Plus 구독(또는 NOW Coins) 잠금** — 비구독자도 유료 호스트가 켠 게임엔 참가 가능 |
| 관전 모드 | 미착석자가 착석자 **전원의 홀카드**를 보는 오픈카드 방식("watching a poker show on TV"). 착석 희망자 전원의 명시 동의 게이트 + 착석자에게 관전자 수 경고 + 관전자가 착석하면 즉시 가시성 상실 |
| 게임 종류 | NLHE 외 PLO 하이, 오마하 하이/로 |

### 수익모델

레이크 없는 프리미엄 + 마이크로트랜잭션 이중 구조:

- **Basic 무료 / Plus $9.99월·$69.60년(41% 할인, 7일 무료 체험) / Platinum $49.99월**
- **NOW Coins**: $4.95(100코인)~$49.95(3,000코인). 공식 아바타·커스텀 펠트·변형 모드 언락.
  Plus 회원은 코인 구매 시 2배(2024-08 공지 기준, 현재 유효성 미재확인).
- Plus 잠금 기능: 변형 룰 활성화, AI 게임 분석 **'Sam Says'**(플레이 스타일 닉네임·플레이어별
  등급·전략 팁 — 실시간 코칭인지 사후 분석인지는 벤더 설명이 모호), 무제한 핸드 히스토리,
  컬러 플레이어 노트, MTT 레이트 레지, 클럽 확장, 광고 숨김.

## 2. 강점

1. **진입 마찰 제로** — 링크 하나로 입장. 포커 도쿠도 localStorage 토큰 + `/?room=` 초대 링크로
   동일 철학이라 이 축은 이미 동등 (우리는 승인 절차 없는 즉시 착석·바이인 슬라이더가 오히려 빠름).
2. **호스트/운영 도구의 깊이** — 30레벨 블라인드, MTT 체크인·초대코드 관리 등 "모임 운영자" 대상
   기능이 탄탄.
3. **홈게임 감성 변형 룰** — run it twice, rabbit hunting, Bomb Pots 등 재미 요소 풍부.
4. **치팅 우려를 설계로 해결한 관전 모드** — 전원 동의 게이트("you will never be cheated").

## 4. 포커 도쿠 벤치마킹 후보 (우선순위 제안)

| 우선순위 | 후보 | 근거 | 구현 비용 감 |
|---|---|---|---|
| P1 | **올인 확인 다이얼로그** (모바일 미스클릭 방지) | 경쟁사 2년째 미해결 요청 | 낮음 |
| P1 | **SnG 블라인드 커스터마이즈** (레벨 수·시간 선택지) | 현재 고정 3분 → blind-schedule.ts 확장 | 낮음 |
| P2 | **관전 모드** — 기본은 마스킹 관전, 옵션으로 전원 동의 오픈카드 2단계 | 동의 게이트 설계 벤치마킹. `getPublicState(forPlayerId)` 마스킹 계약과 자연 정합 | 중간 |
| P2 | **run it twice / rabbit hunting 무료 제공** | 경쟁사가 유료 잠금한 지점 → 그대로 차별화 전환 | 중간 |
| P2 | **정기 모임(클럽)형 SnG** | 경쟁사 1위 미충족 수요를 기존 SnG 자산으로 선점 | 중간~높음 |
| P3 | MTT | 수요 확실하나 인메모리 단일 인스턴스 구조상 장기 과제 (AGENTS.md 미구현 명시) | 높음 |

## 5. 차별화 축 (이미 보유했거나 밀어야 할 것)

- **캐릭터 엔터테인먼트** — 미소녀 딜러+봇 AI+상황 대사는 Poker Now에 없는 정체성.
  Poker Now의 봇/AI 상대 기능은 이번 조사에서 미확인 — 부재가 확정되면 "혼자서도 노는 홀덤"이
  가장 강한 구조적 차별점.
- **빠른 페이싱** — 8초 턴 + 수동 타임칩 + 자리비움/끊김 즉시 자동 액션은 경쟁사의 대표 불만을
  이미 구조적으로 해결한 상태. 소개·마케팅 문구에 명시할 가치.
- **모바일 우선 설계** — 세로형 단일 레이아웃 + 고정 액션 독 + PWA 설치(2026-07-18 추가)가
  경쟁사의 태블릿/모바일 약점을 정면 공략.
- **AI의 방향 차이** — Poker Now는 AI를 분석 도구로 유료 판매(Sam Says), 우리는 캐릭터로 무료
  제공. 직접 충돌하지 않는 축. 장기적으로 핸드 히스토리 + 캐릭터 해설 **'스터디 모드'**
  (dialogue-manager의 situationKey 확장으로 기존 캐시·생성 파이프라인 재사용)로 분석과
  엔터테인먼트를 결합하는 제3의 길 가능.
- **수익화 참고** — 무료 홈게임 시장에서도 편의·연출·분석의 유료화가 성립함을 Poker Now가 증명.
  단 포커 도쿠는 초기에는 전 기능 무료로 유저 확보가 합리적.

2. 실제 유저 규모(MAU/동시접속)와 한국·일본 시장 침투도 — 공개 지표 못 찾음.

   홈게임 그룹 정산 UX는 유력 벤치마킹 후보인데 미조사.
4. 무료(Basic) 티어의 광고/프로모션 노출 수준과 유저 반응.
